Recent Works

Calculation of N point Discrete Fourier transform (DFT) of discrete time sequence.

DFT is used to measure the frequency content of discrete time signal. It provides alternative reprsentaion for discrete time sequence. Also these algorithm is used in Digital filtering to remove unwanted frequencies.

Calculation of Cross correlation between two sequences.

Correlation is signal processing algorithm used to determine how similar two discrete time sequences are. If two sequences are highly correlated(almost similar), there correlation result will be symmetric about n=0

Calculation of Linear convolution between two sequences.

Convolution is basic Signal Processing algorithm used to determine the response of LTI system(h(n)) applied on input signal x(n). These tool is most widely applied algorithm in signal processing application.
